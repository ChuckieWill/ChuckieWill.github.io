小程序中使用npm

小程序使用npm官方文档

使用步骤：

  1. 编辑器右上角：详情–>本地设置–>勾选使用npm模块

  2. 任选一个文件夹右击选择在终端打开

    • cd到项目根目录下
    • 根目录下输入命令：npm init
      • 操作成功后会在项目根目录下生成：package.json文件

  3. 使用npm安装需要的库

    • 在终端根目录下输入安装命令即可：npm install ....

  4. 点击开发者工具中的菜单栏：工具 –> 构建 npm

    • 注意：每次安装依赖后都要点击构建npm
