Git(0x03)-github的使用

发表于 2020-01-19 更新于 2024-09-18 分类于 tools ， git
本文字数： 2.4k 阅读时长 ≈ 2 分钟

1 基本概念

  • 仓库（Repository）:
    • 仓库用来存放代码，每个项目对应一个仓库
  • 收藏（Star）：
    • 收藏项目，方便下次查看
  • 复制克隆项目（Fork）:
    • 将别人的项目克隆到自己的GitHub上
  • 发起请求（Pull Request）:
    • 修改克隆来的项目后 将修改记录发送给克隆来项目的账户 提示对方是否合并修改
  • 关注（Watch）:
    • 关注项目，当项目更新可以接收到通知
  • 事务卡片（Issue）：
    • 发现代码BUG,但是目前没有成型的代码，需要讨论时用
  • GitHub主页
    • 左侧显示自己的仓库
    • 中间显示关注用户的动态
    • 右侧显示发现
