#  bootstrap

> 官方文档：[ https://v5.getbootstrap.com/]( https://v5.getbootstrap.com/)

安装：

```
npm install bootstrap --save
```

引入：

* 在App.vue文件中引入

```vue
import 'bootstrap/dist/css/bootstrap.min.css'
```
