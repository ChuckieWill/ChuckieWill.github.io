#  fastclick

**`fastclick`用于解决移动端点击300ms延迟的问题**

##  基础使用

* [github-fastclick]( https://github.com/ftlabs/fastclick )
* 安装

```
npm install fastclick --save
```

* main.js中导入

```
import FastClick from 'fastclick'
```

* 使用

```js
FastClick.attach(document.body)
```
