#  AntD

**`Ant Design`是适用于`React`的UI组件库**

* **[Ant Design官网]( https://ant.design/index-cn )**

* 安装

  ```js
  npm install antd --save
  ```

* 使用

  * 样式引入(引入到组件的`.js`文件中)

  ```js
  import 'antd/dist/antd.css'; // or 'antd/dist/antd.less'
  ```

  * 在官网查找需要使用的组件并查看使用组件的代码
