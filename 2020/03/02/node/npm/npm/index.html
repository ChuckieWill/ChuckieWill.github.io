npm

1 项目初始化

在开发React、Vue、koa等框架项目时（基于node时），需要初始化项目

  • 命令：npm init

  • 使用前提：终端切换到根目录下

  • 初始化完成后会在根目录下生成package.js文件

2 npm命令

  • –save-dev :安装开发时依赖，项目打包后不需要继续使用的。
  • –save :安装发布时依赖
  • npm uninstall ： 卸载
  • npm install : 将package.js中的库全部安装

3 nrm

安装国内其它镜像源（包括taobao镜像源）

主要使用ls和use命令

  • 1)nrm ls是列出来现在已经配置好的所有的原地址
    • 运行npm ls报错解决方案
  • 2)nrm use是切换到哪个源上
  • 3)nrm add添加源
  • 4)nrm del删除源
  • 5)nrm test测试源的响应时间，可以作为使用哪个源的参考
