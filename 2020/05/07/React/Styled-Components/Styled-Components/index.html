Styled-Components

1 基础

  • styled-components用于管理React样式

    • 默认情况下，react中引入的样式会作用于其所在组件及其所有子组件，这样就会相互影响
    • styled-components的核心功能就是，使组件的样式只作用于组件本身，不作用于其子组件

  • Github地址：styled-components
