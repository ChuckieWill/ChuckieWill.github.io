react-router-dom

react-router-dom是React提供给web端的路由插件，针对React Native有其它的插件

安装：

1 路由定义

在App.js中配置路由

  • <BrowserRouter>: 包裹所有的路由
  • <Route>: 设置具体的路由
    • path: 设置路由的路径
    • exact： 只有路径完全匹配时才会显示
    • component： 配置路由对应的组件

2 单页面路由跳转

2.1 Link方式

单页面跳转使用Link而不用a标签的原因：

  1. a标签跳转会再次刷新页面，再次请求html文件，这样会造成性能的损耗
  2. Link实现跳转时，不会再次请求html文件，这样既节省了性能也满足单页面应用的规则

Link的使用：

  • 路由路径的定义：
    • path: ‘路由路径’
  • 使用Link标签的组件必须被<BrowserRouter>包裹
